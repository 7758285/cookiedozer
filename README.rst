===========
Cookiedozer
===========

`Cookiecutter`_ template for Python `Kivy`_ apps ready to be deployed to android devices with `Buildozer`_.


Usage
-----

Generate a Kivy project::

    cookiecutter https://github.com/hackebrot/cookiedozer
